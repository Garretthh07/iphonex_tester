# What

* iPhone X での見え方をテストするためのフレームを描画します

# Requirement

* Unity 2017.1
* npm

# Install

```shell
$ npm install github:kidsstar/iphonex_tester
```

# Usage

* メニューの `Assets` &gt; `Install iPhone X Tester` からテスターシーンを追加してください。
* 再生中の追加は出来ません。
